12- Crash vs Panic 
Ans: Crash usually happens when a trap occurs when the application trying to access memory incorrectly. 
     Panic usually when the application kill/shutdown itself abruptly. 
     Main difference between crash and panic is that crash is hardware or OS initiated and panic usually imitated by application by calling abort() function.
